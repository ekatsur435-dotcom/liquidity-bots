# 🤖 LIQUIDITY LONG BOT — Проект автоторговли

## 📋 Обзор проекта

Создание системы для автоматической торговли LONG позиций на фьючерсах, зеркальной системе SHORT из DUMP Signals. Бот будет анализировать перепроданные монеты и давать сигналы на LONG с оценкой Long Score.

---

## 🔍 Анализ исходной SHORT системы (из фото)

### Компоненты SHORT Score:

| Параметр | Значение | Логика |
|----------|----------|--------|
| **RSI** | >70 насторожить, >80 экстремум | Перекупленность |
| **SuperTrend** | Отклонение >15% | Аномальное растяжение |
| **Фандинг** | Высокий накопленный | Дорого держать лонг |
| **L/S Ratio** | >65% лонгов | Толпа в одну сторону |
| **OI** | Рост с ценой | Перегруз позиций |
| **Дельта по часам** | Красные часы при росте | Памп выдыхается |
| **Дивергенция** | Цена растёт, дельта падает | Скрытые продажи |
| **Объём по дням** | Памп на 1 дне | Слабая конструкция |

### Паттерны 15m для SHORT:
- 🔴 **REJECTION SHORT** — отбой от уровня
- 🔴 **TRAP LONG** — рывок вверх на медвежьем объёме
- 💥 **MEGA SHORT** — доминация продавцов
- 🟣 **DISTRIBUTION** — крупный игрок сбрасывает

## 🟢 Логика LONG системы (зеркальная)

### Основной принцип: ПОКУПАЙ ПАНИКУ

**Long Score** — итоговая оценка в процентах (65%+ интересно, 75%+ сильный сигнал)

### Компоненты LONG Score:

| Параметр | Значение | Логика | Очки |
|----------|----------|--------|------|
| **RSI 1h** | <30 перепроданность, <20 экстремум | Перепроданность | +15/+20 |
| **SuperTrend** | Отклонение <-10% | Аномальное сжатие | +15 |
| **Фандинг** | Отрицательный (шорты платят лонгам) | Лонгам платят | +10/+15 |
| **L/S Ratio** | <35% лонгов (>65% шортов) | Толпа в шортах | +10/+15 |
| **OI** | Рост ОИ при падении цены | Шорты перегружены | +10/+15 |
| **Дельта по часам** | Зелёные часы при падении цены | Падение выдыхается | +15 |
| **Дивергенция** | Цена падает, дельта положительная | Скрытые покупки | +20 |
| **Объём по дням** | Дамп на 1 дне без продолжения | Слабый дамп | +10 |

### Паттерны 15m для LONG (зеркальные):

| Паттерн | Суть | Сила |
|---------|------|------|
| 🟢 **REJECTION LONG** | Цена отбилась от поддержки, продавцы не удержали, покупатели перехватили | Высокая |
| 🟢 **TRAP SHORT** | Рывок вниз на бычьем объёме, шортисты в ловушке | Высокая |
| 💥 **MEGA LONG** | Покупатели явно доминируют по объёму и импульсу | Очень высокая |
| 🟣 **ACCUMULATION** | Крупный игрок тихо накапливает: тело маленькое, объём огромный, дельта глубоко в плюсе | Максимальная |

## 📊 Детальные паттерны LONG

### 🟢 REJECTION LONG (отбой от поддержки)
**Сигнал:**
- Цена коснулась сильного уровня поддержки (POC, OB, FVG, дневной лоу)
- Бычья свеча с длинным нижним фитилем (отбой)
- Дельта положительная на этом баре
- Объём выше среднего (подтверждение интереса)

**Очки:** +20

### 🟢 TRAP SHORT (ловушка для шортистов)
**Сигнал:**
- Резкий слив ниже уровня (манипуляция)
- Но свеча закрывается выше уровня (бичий engulfing)
- Дельта положительная несмотря на падение
- Большой объём на "откупе"

**Очки:** +15/+20 (в зависимости от качества)

### 💥 MEGA LONG (мега доминация покупателей)
**Сигнал:**
- 4-5 зелёных часов подряд
- Дельта положительная на каждом часе
- Объём растёт с каждым часом
- Цена растёт или стабилизируется
- Дамп закончился, начался откуп

**Очки:** +25

### 🟣 ACCUMULATION (аккумуляция крупного игрока)
**Сигнал:**
- Цена боковит или плавно падает
- Объём огромный (в 2-3x выше среднего)
- Дельта сильно положительная (+5% и выше)
- Маленькие тела свечей (боковик)
- Похоже на Wyckoff Spring/SoS

**Очки:** +25/+30

### 2. Score Calculation Algorithm

```python
def calculate_long_score(symbol_data):
    score = 0
    max_score = 100
    
    # RSI Component (max 20 points)
    if rsi_1h < 20:
        score += 20  # Экстремальная перепроданность
    elif rsi_1h < 30:
        score += 15  # Перепроданность
    elif rsi_1h < 40:
        score += 8   # Начало перепроданности
    
    # Funding Component (max 15 points)
    if funding_accumulated < -0.5:  # Шорты платят сильно
        score += 15
    elif funding_accumulated < -0.2:
        score += 10
    
    # L/S Ratio Component (max 15 points)
    if long_ratio < 30:  # 70%+ шортов
        score += 15
    elif long_ratio < 40:
        score += 10
    
    # OI Component (max 15 points)
    if oi_growing and price_falling:
        score += 15  # Шорты набирают, скоро сожмут
    
    # Delta Component (max 20 points)
    if divergence_bull:  # Цена падает, дельта растёт
        score += 20
    elif positive_delta_hours >= 4:
        score += 15
    
    # Pattern Component (max 25 points)
    if pattern_accumulation:
        score += 25
    elif pattern_mega_long:
        score += 20
    elif pattern_trap_short:
        score += 15
    elif pattern_rejection:
        score += 10
    
    return min(score, max_score)
```

### 3. Pattern Detection Algorithm

```python
def detect_patterns(ohlcv_data, delta_data, volume_profile):
    patterns = []
    
    # MEGA LONG Detection
    recent_candles = ohlcv_data[-5:]  # последние 5 часов
    green_hours = sum(1 for c in recent_candles if c['close'] > c['open'])
    positive_delta = sum(1 for d in delta_data[-5:] if d > 0)
    
    if green_hours >= 4 and positive_delta >= 4:
        volume_trend = all(volume_profile[-i] > volume_profile[-i-1] 
                          for i in range(1, 5))
        if volume_trend:
            patterns.append(('MEGA_LONG', 25))
    
    # TRAP SHORT Detection
    last_candle = ohlcv_data[-1]
    prev_candle = ohlcv_data[-2]
    
    if (last_candle['low'] < prev_candle['low'] * 0.99 and  # Пробой лоя
        last_candle['close'] > prev_candle['open'] and       # Но закрытие выше открытия пред.
        delta_data[-1] > 0):                                  # Положительная дельта
        patterns.append(('TRAP_SHORT', 20))
    
    # ACCUMULATION Detection
    avg_volume = sum(c['volume'] for c in ohlcv_data[-20:-5]) / 15
    recent_volume = sum(c['volume'] for c in ohlcv_data[-5:]) / 5
    
    if (recent_volume > avg_volume * 2.5 and  # Объём в 2.5x выше
        abs(last_candle['close'] - last_candle['open']) / last_candle['close'] < 0.005 and  # Маленькое тело
        sum(delta_data[-5:]) > 0.05):  # Сильная положительная дельта
        patterns.append(('ACCUMULATION', 30))
    
    return patterns
```

## 🔒 Безопасность и риски

### Риски системы:

1. **Ложные сигналы** — дамп может продолжиться
   - *Митигация:* Ждать подтверждения отскока (1-2 свечи)
   
2. **Ловушка медведей** — цена продолжит падение после входа
   - *Митигация:* Строгий SL, не более 1% риска на сделку
   
3. **Низкая ликвидность** — на альткоинах слиппедж
   - *Митигация:* Торговать только топ-50 монет по объёму
   
4. **Волатильность** — резкие движения пробивают SL
   - *Митигация:* ATR-based SL, адаптивный к волатильности

### Рекомендуемые максимумы:
- Макс. 5 позиций одновременно
- Макс. плечо 5x для LONG (ниже чем для SHORT)
- Макс. риск 1% на сделку
- Макс. риск 3% на все позиции
