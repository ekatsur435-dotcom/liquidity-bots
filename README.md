## 🎯 Обзор

**Liquidity Long Bot** — это система для автоматического поиска и торговли перепроданными фьючерсными парами на криптобиржах (Binance Futures, Bybit).

### Основной принцип: **ПОКУПАЙ ПАНИКУ**

Когда толпа в панике продаёт, а шортисты перегружают позиции — это лучшее время для входа в LONG.

### Ключевые особенности:

- **Long Score (0-100%)** — итоговая оценка силы сигнала
- **4 паттерна на 15m** — детекция оптимальных точек входа
- **Анализ микроструктуры** — RSI, фандинг, OI, L/S ratio, дельта
- **Авто-торговля** — интеграция с Bybit API
- **Risk Management** — Smart SL, 6 уровней TP, трейлинг-стоп
- **Telegram уведомления** — мгновенные алерты

## 🧠 Логика системы

### Long Score — из чего состоит:

| Компонент | Макс очков | Что измеряет |
|-----------|------------|--------------|
| **RSI 1h** | 20 | Перепроданность (ниже 30 = хорошо) |
| **Фандинг** | 15 | Плата шортистов (отрицательный = хорошо) |
| **L/S Ratio** | 15 | Соотношение лонгов/шортов (<35% лонгов = хорошо) |
| **Open Interest** | 15 | Рост OI при падении = шорты накапливаются |
| **Дельта** | 20 | Покупательское давление |
| **Паттерны** | 30 | Технические паттерны на 15m |

### Уровни сигналов:

| Score | Уровень | Действие |
|-------|---------|----------|
| < 65% | Слабый | Игнорировать |
| 65-74% | Интересный | Можно рассмотреть |
| 75-84% | Сильный | Рекомендуется вход |
| 85%+ | Экстремальный | Высокая вероятность успеха |

### Паттерны для LONG:

| Паттерн | Суть | Очки |
|---------|------|------|
| 🟢 **REJECTION LONG** | Отбой от поддержки | 15 |
| 🟢 **TRAP SHORT** | Ловушка для шортистов | 20 |
| 💥 **MEGA LONG** | Доминация покупателей | 25 |
| 🟣 **ACCUMULATION** | Накопление крупного игрока | 30 |
